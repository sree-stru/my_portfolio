# Portfolio (Vite + React)

This is a minimal responsive React portfolio scaffold using Vite.

## Setup (PowerShell on Windows)

```powershell
cd d:\portfolio_final
npm install
npm run dev
```

- `npm run dev` starts the dev server (Vite).
- `npm run build` produces a `dist` folder for production.
- `npm run preview` serves the built `dist` locally for a final check.

## Header behavior implemented
- Large single-line uppercase name: `ANUSREE K JINAN`.
- Name scales responsively to fit the header and keeps horizontal padding.
- `graphic designer` rendered below in red cursive (`Allura`).
- Gradient divider below the header.

## Files
- `index.html` — app entry with Google Fonts.
- `src/main.jsx` — React entry file.
- `src/App.jsx` — header + simple content and fit-text logic.
- `src/index.css` — responsive styles.

Customize `src/App.jsx` and `src/index.css` with your real content, images, and project items.
